# Design System - BETAPREDIT

## Icon System
- **Library**: lucide-react (Professional SVG icons)
- **Component**: `Icon` from `components/icons/IconSystem`
- **Usage**: Replace all emojis with professional icons

## Color Palette
- **Primary**: Indigo/Purple gradients (professional, trustworthy)
- **Accent**: Cyan/Teal (energy, action)
- **Gold**: Success, premium features
- **Background**: Dark slate-900 (modern, easy on eyes)

## Design Principles
1. **Professional over Generic**: No emojis, use SVG icons
2. **Consistent Spacing**: Use Tailwind spacing scale
3. **Modern Gradients**: Subtle, professional gradients
4. **Glass Morphism**: Backdrop blur effects for depth
5. **Smooth Animations**: Hover states, transitions
6. **Responsive First**: Mobile-optimized layouts

## Component Patterns
- Cards: Rounded-xl, subtle borders, hover effects
- Buttons: Gradient backgrounds, smooth transitions
- Icons: 20-24px default, consistent stroke width
- Typography: Inter font, clear hierarchy
